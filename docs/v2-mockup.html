3. Design Decisions & References

This v2.0 reskin takes inspiration from WindUI (Mac topbar, OpenButton with Draggable, Group/Tag/Space), Maclib (sectioned layout, slider+value), Modern Sirius (header + info cards), and Abyss Script (toggle rows with description, slider with value badge). Combined into a single, opinionated design that matches the user's brand.

Window & Topbar

  • Mac-style traffic lights (red/yellow/green)
  • Drag handle: the entire topbar (or anywhere outside elements)
  • Title + subtitle centered
  • Tag pills (v2.0, Premium) right of title
  • Theme picker (4 swatches) far right

Tabs & Search

  • Icon + label, underline indicator on active
  • Real-time search filters tabs
  • Accent glow on active tab indicator
  • Smooth color transition on hover/active

Components

  • Toggle (Switch) + Checkbox variant
  • Slider with value badge (right-aligned)
  • Dropdown with single + multi-select
  • Input field + Textarea
  • Colorpicker + Keybind
  • Image + Paragraph (rich content)

FAB (OpenButton)

  • Floating bottom-right, gradient fill
  • Glow halo + 4px inner shadow
  • Tap → open menu
  • Hold → drag to reposition (8pt threshold)
  • Snap to edges on release

Status bar

  • FPS (emerald), Ping (blue), Active count (gold)
  • STOP panic button (red, glow on hover)
  • Live updating every 500ms
  • Always visible at bottom

Theme presets

  • Emerald (default, your color)
  • Amethyst (cosmic purple)
  • Sapphire (cool blue)
  • Rose (hot pink)
  • All tokens swap in real time
